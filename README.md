# EduTest Platform

EduTest is an online testing and education management platform built with Python, Flask, and PostgreSQL.

## Features

- User authentication system
- Admin, teacher, and student roles
- Test creation and management
- Question import from TXT, DOCX, and XLSX
- Result tracking and statistics
- PostgreSQL database support
- Flask-Migrate/Alembic migrations
- Anti-cheat activity tracking
- Responsive dark UI

## Technologies Used

- Python
- Flask
- PostgreSQL
- SQLAlchemy
- Flask-Migrate
- HTML/CSS
- JavaScript
