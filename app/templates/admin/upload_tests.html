  • Word (.docx): формат блоков. Пример: 1 ballik savollar, затем строки Savol:, Turi:, Javoblar: или Javob:. Картинку можно вставить сразу после строки вопроса.
  • Excel (.xlsx): поддерживается и сам Excel, и ZIP-архив с Excel + картинками.
  • ZIP (.zip): внутри должен быть один файл .xlsx и папка images с картинками.
  • TXT: старый текстовый формат по-прежнему поддерживается.

Для Word блочный балл работает до следующего заголовка вроде 2 ballik savollar или 3 балла. Для Excel значение из колонки block_points действует на следующие строки, пока не встретится новое.

Для Excel используй колонки: block_points, question, image_name, type, answer1, answer2, answer3, answer4, correct.

Для вопросов с одним правильным ответом в Excel укажи в колонке correct номер правильного варианта, например 2. Для нескольких правильных ответов — через запятую, например 1,3.
